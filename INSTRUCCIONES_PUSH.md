# Instrucciones para desbloquear el push

1. Ve a este enlace en tu navegador:
   https://github.com/juandi251/FriendlyVoice-Back/security/secret-scanning/unblock-secret/34rhpWbvl9Wtk7ymVrYPWuCTUBH

2. Haz clic en "Allow secret" o "Permitir secret"

3. Repite para el segundo enlace:
   https://github.com/juandi251/FriendlyVoice-Back/security/secret-scanning/unblock-secret/34rhpTXe2gN433RAxRcfopg7gZw

4. Luego ejecuta en tu terminal:
   ```bash
   git push origin main
   ```

Esto permitirá que los cambios se desplieguen en Render automáticamente.
